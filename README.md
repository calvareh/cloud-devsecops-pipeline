A hands-on DevSecOps project focused on implementing automated security gates within a CI/CD pipeline using GitHub Actions and open-source security tools.

## Project Goals

- Implement automated SAST scanning
- Perform dependency vulnerability scanning
- Scan Infrastructure as Code (IaC)
- Scan container images for vulnerabilities
- Detect exposed secrets
- Enforce pass/fail security gates
- Demonstrate secure SDLC practices

## Technologies

- GitHub Actions
- Docker
- Semgrep
- Trivy
- Checkov
- Gitleaks
- Python Flask

## Planned Security Gates

| Security Control | Tool |
|---|---|
| SAST | Semgrep |
| Secrets Detection | Gitleaks |
| Dependency Scanning | Trivy |
| IaC Scanning | Checkov |
| Container Scanning | Trivy |
